Using the place endpoint

Here, locations for events are listed. Events in each location can be found at the event endpoint using the query parameter location. Locations are added from Google Maps by the users.

Place text

To find places that contain a specific string, use the query parameter text.

Example:

place/?text=tori

See the result

Ordering

Default ordering is decreasing order by the number of events found. You may also order results by name, street_address or postal_code. For example:

place/?sort=name

See the result

Showing all places

By default, only those locations which have listed events are displayed. You may display locations with zero n_events with the query parameter show_all_places.
